# AI提示词库

## 使用原则
1. **明确具体**：越具体，回答质量越高
2. **分步骤**：复杂问题分解为多个简单问题
3. **提供上下文**：让AI了解你的项目和进度
4. **指定格式**：明确期望的输出格式
5. **迭代优化**：基于结果调整提示词

## 提示词优化记录

### 有效提示词特征
1. **具体明确**：而不是"帮我写个API"
2. **上下文充分**：说明项目背景和进展
3. **格式要求**：指定输出格式和结构
4. **技术要求**：明确技术栈和约束
5. **分步骤**：复杂问题分解

### 无效提示词特征
1. **过于宽泛**："教我编程"
2. **缺少上下文**：不说明已有工作
3. **矛盾要求**：同时要求简单和完整
4. **不切实际**：短时间内完成复杂功能
5. **格式混乱**：没有明确的结构要求
