|------|-----|
| **需求ID** | FR-001 |
| **优先级** | P0 |
| **状态** | ✅ 已实现 |

**描述**：配置pytest测试运行环境

**验收标准**：
- [x] AC-001.1: pytest.ini配置文件
- [x] AC-001.2: 测试路径 `tests/`
- [x] AC-001.3: 自动覆盖率报告（term + html）
- [x] AC-001.4: 依赖：pytest==7.4.3, pytest-cov==4.1.0

**配置规格**：
```ini
[pytest]
testpaths = tests
python_files = test_*.py
python_classes = Test*
python_functions = test_*
addopts = -v --cov=. --cov-report=term-missing --cov-report=html
```
