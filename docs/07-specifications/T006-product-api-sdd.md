## 1. 概述

### 1.1 目的
定义产品管理的RESTful API端点，实现完整的CRUD操作。

### 1.2 范围
- 产品列表查询（分页+过滤）
- 单产品查询
- 产品创建（去重校验）
- 产品更新
- 产品删除

---

## 2. 功能需求规格

### FR-001: 产品列表查询

| 属性 | 值 |
|------|-----|
| **需求ID** | FR-001 |
| **优先级** | P0 |
| **状态** | ✅ 已实现 |

**描述**：`GET /products/` 分页查询产品列表，支持分类和状态过滤

**验收标准**：
- [x] AC-001.1: 分页参数 `skip`（默认0, ≥0）和 `limit`（默认100, 1-1000）
- [x] AC-001.2: 可选过滤参数 `category`、`status`
- [x] AC-001.3: 返回 `List[Product]`
- [x] AC-001.4: 200响应码

**接口规格**：
```
GET /products/?skip=0&limit=100&category=Electronics&status=draft
```
