### 解决的技术问题
1. **SQLite UUID兼容性**
   - 问题：SQLite不支持PostgreSQL UUID类型

   - 问题：测试环境无真实JWT认证
   - 解决：通过 `app.dependency_overrides` 注入测试用户（admin角色）

3. **状态枚举验证**
   - 问题：`status: str` 接受任意字符串，无验证
   - 解决：使用 `Literal["draft","active","released","archived","obsolete"]` 类型约束
